# Security

If you discover a security vulnerability in Detto, please report it privately.

**Contact:** jason@gremble.io

**Guidelines:**
- Do not open a public issue for security vulnerabilities
- Include steps to reproduce and any relevant details about your environment
- I will acknowledge receipt within 48 hours
- I will work on a fix and coordinate disclosure timing with you
- Please allow a reasonable window (up to 90 days) before public disclosure
